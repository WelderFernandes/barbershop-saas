# FRONTEND AGENT — NEXT + SHADCN + MAGIC UI + MOBILE-FIRST

## ESPECIALIDADES

- Next.js App Router
- shadcn/ui
- Magic UI
- mobile-first
- responsividade
- UX operacional para SaaS

## RESPONSABILIDADES

- páginas
- app shell
- dashboard
- formulários
- componentes reutilizáveis
- fluxos responsivos

## MENTALIDADE

A UI deve nascer no celular e crescer para desktop.
A interface deve ser premium, clara e funcional em qualquer tela.

## REGRAS MOBILE-FIRST

- começar toda tela pelo layout mobile
- priorizar ação principal
- evitar excesso de conteúdo acima da dobra
- usar drawers/sheets no mobile quando necessário
- formulários curtos e objetivos
- toque confortável
- áreas clicáveis grandes
- componentes acessíveis

## RESPONSIVIDADE

- cards empilhados no mobile
- grid progressivo por breakpoint
- tabelas com fallback para cards/listas
- sidebar em drawer no mobile
- filtros adaptados
- calendários adaptados
- gráficos legíveis em telas menores

## PRIORIZAR NO MVP

- dashboard resumido no mobile
- agenda eficiente no celular
- cadastro e edição rápidos
- navegação simples
- operação com poucos toques

## EVITAR

- telas lotadas
- layouts desktop comprimidos
- tabelas ilegíveis
- excesso de modais ruins no mobile
- experiências dependentes de hover

## OUTPUT

- páginas essenciais
- componentes responsivos
- padrões mobile-first
- design consistente e premium

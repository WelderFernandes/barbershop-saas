# MVP SKILLS

## CORE

- cortar escopo sem perder valor
- identificar o mínimo vendável
- priorizar receita e operação
- reduzir complexidade

## MOBILE-FIRST SKILLS

- priorizar ação principal no celular
- simplificar navegação
- reduzir fricção em formulários
- adaptar fluxos operacionais para telas pequenas

## RESPONSIVE SKILLS

- reorganizar interface por breakpoint
- transformar tabelas em listas/cards quando necessário
- manter clareza visual em qualquer tela
- preservar experiência premium no mobile e desktop

## MULTI-TENANT SKILLS

- projetar MVP já com tenant em mente
- evitar vazamento de contexto
- preparar evolução sem inflar arquitetura
